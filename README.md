# 🎨 无限画布 (Infinite Canvas)

一个功能强大的 2D 无限画布创意工作台，支持多种节点类型、贝塞尔曲线串线和姿势骨骼系统。

**在线体验**: https://your-netlify-url.netlify.app

---

## ✨ 功能特性

### 🎯 核心功能
| 功能 | 说明 |
|------|------|
| **无限画布** | 支持缩放（8%-400%）、平移操作，滚轮缩放 + 空格拖拽 |
| **6种节点** | 文本、图片、视频、姿势、图集、涂鸦 |
| **贝塞尔串线** | SVG 贝塞尔曲线连接，支持拖拽调整控制点 |
| **姿势骨骼** | 单人/多人骨骼显示，支持肢体拖拽调整 |
| **素材库** | 内置素材管理，支持添加本地素材 |
| **导出功能** | 支持 PNG/JPG 导出画布内容 |

### 🖱️ 操作方式
| 操作 | 功能 |
|------|------|
| 滚轮 | 缩放画布 |
| 长按空白区域 | 平移画布 |
| 空格 + 拖拽 | 临时手型平移 |
| 左键拖拽节点 | 移动节点 |
| 双击节点 | 编辑内容 |
| 右键节点 | 打开快捷菜单 |

### 📝 节点类型
- **📝 文本节点** - 富文本编辑，支持基础格式
- **🖼️ 图片节点** - 支持本地/网络图片，拖拽上传
- **🎬 视频节点** - 支持本地/网络视频播放
- **🏃 姿势节点** - 火柴人骨骼可视化，支持多种预设
- **📚 图集节点** - 展示多张图片的网格布局
- **✏️ 涂鸦节点** - 自由绘制画布

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                     无限画布架构                          │
├─────────────────────────────────────────────────────────┤
│  UI Layer                                               │
│  ├── index.html (入口 + 布局)                           │
│  ├── styles/ (CSS 样式)                                 │
│  │   ├── main.css (画布 + 工具栏)                       │
│  │   ├── nodes.css (节点样式)                           │
│  │   └── panels.css (侧边栏/面板)                       │
│  └── 组件层                                              │
│      └── 7种节点类型 (HTML渲染)                          │
├─────────────────────────────────────────────────────────┤
│  Logic Layer (JavaScript ES6+)                          │
│  ├── canvas.js (画布引擎 - 缩放/平移/坐标变换)            │
│  ├── connections.js (贝塞尔曲线连接管理)                 │
│  ├── library.js (素材库管理)                             │
│  ├── utils.js (工具函数)                                │
│  └── app.js (应用入口 + 事件协调)                        │
├─────────────────────────────────────────────────────────┤
│  Node System                                            │
│  ├── BaseNode.js (节点基类 - 通用生命周期)                │
│  ├── TextNode.js (文本节点)                             │
│  ├── ImageNode.js (图片节点)                            │
│  ├── VideoNode.js (视频节点)                            │
│  ├── PoseNode.js (姿势骨骼节点 - 核心特色)               │
│  ├── GalleryNode.js (图集节点)                          │
│  └── DoodleNode.js (涂鸦节点)                           │
└─────────────────────────────────────────────────────────┘
```

### 关键技术点

| 模块 | 技术实现 |
|------|----------|
| **画布变换** | CSS transform (translate + scale)，支持任意缩放 |
| **串线系统** | SVG `<path>` + 贝塞尔曲线算法，C1 连续 |
| **坐标系统** | screenToCanvas / canvasToScreen 双向往返变换 |
| **节点通信** | 自定义事件 + Connections 模块统一管理 |
| **数据持久化** | LocalStorage 保存画布状态 |

## 🚀 快速开始

### 方式一：直接打开
下载后双击 `dist/index.html` 即可使用（无需安装任何依赖）。

### 方式二：本地服务器
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# 然后访问 http://localhost:8080
```

### 方式三：部署到云端
1. **Netlify Drop**: 直接拖拽 `dist` 文件夹到 https://app.netlify.com/drop
2. **GitHub Pages**: 连接仓库，设置输出目录为 `dist`
3. **Vercel**: `vercel --prod`

## 🎭 姿势节点使用指南

姿势节点是本项目的核心特色功能：

### 模式切换
| 人数 | 模式 | 功能 |
|------|------|------|
| 1-7人 | 肢体调整模式 | 可拖拽调整每个火柴人的肢体关节 |
| 8-9人 | 网格模式 | 整齐排列展示 |
| ≥10人 | 军阵模式 | 大规模人群编排 |

### 预设动作
- **站姿** - 标准站立
- **行走** - 走路姿势
- **奔跑** - 跑步姿势
- **格斗** - 出拳姿势
- **打坐** - 坐姿

### 调整操作
1. 点击选中姿势节点
2. 点击身体关节（圆点）拖拽调整
3. 每人独立一套骨骼数据，互不影响

## 🛠️ 开发说明

### 环境要求
- 浏览器：Chrome 90+ / Firefox 88+ / Safari 14+ / Edge 90+
- 无需 Node.js（纯前端项目）

### 构建
如需修改源码：
1. 编辑 `js/` 和 `styles/` 目录下的文件
2. 将修改后的文件同步到 `dist/` 对应位置
3. 重新部署

### 调试
打开 `dist/debug.html` 可进入调试模式。
